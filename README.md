# Resume RAG AI

AI-powered Resume Intelligence Platform built using:

- React
- FastAPI
- Gemini API
- ChromaDB
- LangChain
- RAG Architecture

## Features

- Resume Upload
- AI Resume Chat
- ATS Score Analysis
- Semantic Resume Search
- Modern UI
- Vector Embeddings

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- FastAPI
- LangChain
- Gemini
- ChromaDB
